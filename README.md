# Présentation du projet

## Informations générales

Ce dépôt contient une application web moderne développée avec des technologies front-end récentes.  
Le projet peut être modifié, exécuté et déployé localement ou via des plateformes de déploiement standards.

---

## Comment modifier le code ?

Plusieurs méthodes sont possibles pour travailler sur ce projet.

### Utiliser un IDE local

Vous pouvez cloner le dépôt et travailler localement avec votre éditeur favori (VS Code, WebStorm, etc.).

**Prérequis :**
- Node.js
- npm  

(Il est recommandé d’installer Node.js via nvm)

Étapes :

```sh
# Étape 1 : Cloner le dépôt
git clone <URL_DU_DEPOT_GIT>

# Étape 2 : Accéder au dossier du projet
cd <NOM_DU_PROJET>

# Étape 3 : Installer les dépendances
npm install

# Étape 4 : Lancer le serveur de développement
npm run dev
